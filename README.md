# vue-cprogress
Simple circle progress bar

[![NPM](https://nodei.co/npm/vue-cprogress.png?compact=true)](https://nodei.co/npm/vue-cprogress/)

### Install

```bash
npm install --save vue-cprogress
```

### Use

```html
<c-progress :total="10" :completed="5"></c-progress>
```

### Options

* diameter
* color
* total
* completed
* width
